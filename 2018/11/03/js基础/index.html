JS的三种引入方式

  • 直接在时间属性中写js代码
  • 在script标签中写js代码

js和java的区别

  • 编译型语言：强类型的语言，先编译后运行
  • 解释型语言：弱类型的语言，直接运行

方法

在js中方法也是对象，实际上js内置了一些数据类型：Math、String、Array、Function

API

全局的函数：属window的函数，因为window代指的是整个浏览器对象，所以整个浏览器中就只有一个window，没有歧义，所以在调动全局函数或者全局变量的时候就可以省略window

  • onload: 当加载完毕dom树的时候执行，在html元素上声明
